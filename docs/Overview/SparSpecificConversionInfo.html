Spar Specific Conversion Information

  1. Use a -include menu.svg to include the menu into all the converted SVGs.
  2. Use 
    :add svg { 
    "xmlns:dxf2svg" = "http://www.l-3com.com/svgns";
    }.
  3. From now on all illustrations converted with Dxf2Svg will have a super element (below the root svg element) which groups all the elements into a page. Example:
    <svg>
    <g id="Page_1">
    <... elements...>
    </g>
    </svg>
  4. There is a menu.svg which is merged into all the finished SVGs. This is done through in step 1, but when it is done an additional element is calculated and added to the converted SVG automatically at conversion time, and it looks like this: <dxf2svg:init height="450.34" width="778.97" activatePalette="true"/> if the illustration is a wiring diagram, and if it is not, activatePalette is set to false. This detection occurs automatically, and switches off the colour selection palette.
  5. Wire runs are converted from lines, arcs or any other SvgDoubleEndedGraphicElements into <path>s. Extra attributes can be applied via the config.d2s by :adding to either 'path' or 'withid' of 'wire_run' which is a predifined id attribute of wires.
